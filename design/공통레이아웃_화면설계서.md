## 1. 전체 레이아웃 구조

```
┌──────────────────────────────────────────────────────┐
│                    TOPBAR (56px)                      │ ← 고정
├──────────┬───────────────────────────────────────────┤
│          │                                           │
│  SIDEBAR │              MAIN CONTENT                 │
│  (200px) │                                           │
│  고정     │         스크롤 가능 영역                   │
│          │                                           │
│  [하단]   │                                           │
│  컨텍스트 │                                           │
│  선택기   │                                           │
└──────────┴───────────────────────────────────────────┘
```

- TOPBAR: 상단 고정 (position: fixed / sticky)
- SIDEBAR: 좌측 고정 (position: fixed), TOPBAR 아래부터 화면 끝까지
- MAIN CONTENT: 나머지 영역, 독립적으로 스크롤

## 2. TOPBAR

### 구성 요소

```
┌──────────────────────────────────────────────────────────────┐
│ 🛡️ 안전NOW          [알림 🔔 N]       [박안전 ∨] 다온산업 본사 │
│ (로고)              (알림 아이콘)      (프로필 드롭다운)        │
└──────────────────────────────────────────────────────────────┘
```

### 스펙

| 항목     | 값                 |
| -------- | ------------------ |
| 높이     | 56px               |
| 배경색   | `#1E2A3B` (네이비) |
| position | fixed top          |
| z-index  | 100                |

### 구성 요소 상세

#### 로고 (좌측)

| 요소      | 내용                                                 |
| --------- | ---------------------------------------------------- |
| 아이콘    | 🛡️ 또는 SVG 로고                                     |
| 텍스트    | 안전NOW                                              |
| 크기      | 아이콘 32px, 텍스트 16px font-weight:800             |
| 색상      | 흰색                                                 |
| 클릭 동작 | 현재 컨텍스트 기준 기본 화면으로 이동 (DSH 또는 TSK) |

#### 알림 아이콘 (우측 영역 좌측)

| 요소        | 내용                                  |
| ----------- | ------------------------------------- |
| 아이콘      | 🔔                                    |
| 미읽음 배지 | 미읽음 알림 수 표시 (0이면 배지 숨김) |
| 최대 표시   | 99+                                   |
| 클릭 동작   | 알림 드롭다운 패널 오픈 (아래 참조)   |

**알림 드롭다운 패널**

```
┌────────────────────────────────────────┐
│ 알림                    [모두 읽음 처리] │
│────────────────────────────────────────│
│ 🔵 [서명요청] 안전경영방침 서명 요청     │
│    2026-04-30 15:23                    │
│────────────────────────────────────────│
│ 🔵 [결재요청] 2026년 예산 결재 요청     │
│    2026-04-29 10:10                    │
│────────────────────────────────────────│
│ ○  [완료] 개선조치 조치 완료            │
│    2026-04-28 09:00                    │
│────────────────────────────────────────│
│          [알림 전체보기 →]              │
└────────────────────────────────────────┘
```

- 최근 10건 표시
- 미읽음: 파랑 도트 + 진한 배경
- 읽음: 회색 도트 + 기본 배경
- 항목 클릭 → 해당 화면으로 이동 + 읽음 처리
- [알림 전체보기] → MYI03-L

#### 프로필 드롭다운 (우측)

```
[사용자 아바타]  박안전   다온산업 본사
```

- 아바타: 이름 첫 글자 원형 배지 (32px)
- 이름: `users.name`
- 현재 사업장: `session.context.company_name` + `site_name`
- 클릭 시 드롭다운:

```
┌──────────────────────────┐
│ 박안전                    │
│ ceo@daon.com             │
│ 다온산업 본사 · 경영책임자  │
│──────────────────────────│
│ 👤 내 정보관리             │ → MYI01-S
│ 🏢 사업장 전환             │ → LOGIN-S
│──────────────────────────│
│ 로그아웃                  │
└──────────────────────────┘
```

## 3. SIDEBAR (GNB)

### 구성 요소

```
┌──────────────────┐
│  [메뉴 항목들]    │ ← 상단 메뉴 목록
│                  │
│  ...             │
│                  │
│ ────────────────│
│  [컨텍스트 선택기] │ ← 하단 고정
└──────────────────┘
```

### 스펙

| 항목       | 값                                        |
| ---------- | ----------------------------------------- |
| 너비       | 200px                                     |
| 배경색     | `#1E2A3B` (네이비)                        |
| position   | fixed left, TOPBAR 아래 (top: 56px)       |
| 높이       | calc(100vh - 56px)                        |
| 오버플로우 | 메뉴 영역: scroll / 컨텍스트 선택기: 고정 |

### 메뉴 항목 구성

```
┌──────────────────┐
│ 📊 대시보드       │ ← 활성 (하이라이트)
│ ✅ 내 업무    [5] │ ← 배지 (미완료 건수)
│ ⚠️ 위험성평가     │
│ 🚌 TBM           │
│ 🔍 안전점검       │
│ 📚 안전보건교육   │
│ 📋 안전경영방침   │
│ 💰 안전보건예산   │
│ 💬 의견청취   [2] │
│ 🔧 개선조치   [8] │
│ 📅 이행관리       │
│ 📁 업무문서관리   │
│ ⚙️ 공정관리       │
│ 🤝 도급관리       │
│ 🏢 사업장관리     │
│ ❓ 고객지원       │
│ 👤 내 정보관리    │
└──────────────────┘
```

### 메뉴 항목 스펙

| 상태 | 배경      | 텍스트    | 아이콘    |
| ---- | --------- | --------- | --------- |
| 기본 | 투명      | `#9CA3AF` | `#9CA3AF` |
| 호버 | `#2D3F56` | `#E5E7EB` | `#E5E7EB` |
| 활성 | `#2563EB` | `#fff`    | `#fff`    |

**배지 (카운트)**

- 내 업무, 의견청취, 개선조치 등 미처리 건수 표시
- 배지 색상: 빨강 (`#DC2626`)
- 최대 표시: 99+
- 0이면 숨김

### 메뉴 그룹 구분선 (선택)

메뉴 수가 많아서 그룹 구분선으로 시각적 분리 가능.

```
│ ────────────────│  ← 안전 활동
│ ⚠️ 위험성평가     │
│ 🚌 TBM           │
│ 🔍 안전점검       │
│ 📚 안전보건교육   │
│ ────────────────│  ← 관리
│ 📋 안전경영방침   │
│ 💰 안전보건예산   │
│ 💬 의견청취       │
│ 🔧 개선조치       │
│ 📅 이행관리       │
│ 📁 업무문서관리   │
│ ⚙️ 공정관리       │
│ ────────────────│  ← 조직/계약
│ 🤝 도급관리       │
│ 🏢 사업장관리     │
│ ────────────────│  ← 지원
│ ❓ 고객지원       │
│ 👤 내 정보관리    │
```

## 4. 컨텍스트 선택기 (SIDEBAR 하단 고정)

로그인 후 어느 사업장 컨텍스트에서 작업 중인지 표시하고, 전환 가능.

### 구성

```
┌──────────────────────┐
│ 🏢 다온산업           │
│    본사 사업장        │
│    경영책임자     ∨   │ ← 클릭 시 LOGIN-S로 이동
└──────────────────────┘
```

- 상단 구분선
- 회사 아이콘 + 회사명
- 사업장명 (sites.name)
- 역할 + 전환 아이콘(∨)
- 클릭 → LOGIN-S (사업장 선택 화면)로 이동

### 도급업체/컨설팅업체 컨텍스트인 경우

```
┌──────────────────────┐
│ [도급업체]            │ ← 주황 배지
│ 동방기계㈜            │
│ 다온산업 본사 현장    │
│ 도급업체 담당자   ∨   │
└──────────────────────┘
```

## 5. 컨텍스트별 GNB 메뉴 노출 규칙

### 5-1. 구독사업자 — CEO / GM / SM / SHM

17개 전체 메뉴 노출. 배지(카운트) 포함.

```
대시보드 / 내 업무 / 위험성평가 / TBM / 안전점검 /
안전보건교육 / 안전경영방침 / 안전보건예산 / 의견청취 /
개선조치 / 이행관리 / 업무문서관리 / 공정관리 /
도급관리 / 사업장관리 / 고객지원 / 내 정보관리
```

**역할별 세부 차이**

| 메뉴         |   CEO   |   GM    |        SM        |       SHM        |
| ------------ | :-----: | :-----: | :--------------: | :--------------: |
| 대시보드     | DSH01-V | DSH02-V |     DSH02-V      |     DSH02-V      |
| 안전보건예산 | ✅ 전체 | ✅ 전체 |  ✅ 본인 사업소  |  ✅ 본인 사업소  |
| 사업장관리   | ✅ 목록 | ✅ 목록 | ✅ 본인 사업소만 | ✅ 본인 사업소만 |

### 5-2. 구독사업자 — WKR (근로자)

```
대시보드 (DSH03-V) / 내 업무 / TBM /
의견청취 / 고객지원 / 내 정보관리
```

총 6개 메뉴. 카운트 배지는 내 업무만.

### 5-3. 도급업체 (SUB)

```
내 업무 / TBM / 의견청취 / 고객지원 / 내 정보관리
```

총 5개 메뉴.

- 대시보드 없음 → 기본 진입 화면은 DSH04-V (간소 대시보드, 별도 화면)
- TBM: 자사 TBM만 / 스케줄·그룹 탭 숨김

### 5-4. 컨설팅업체 (CON)

```
내 업무 / 고객지원 / 내 정보관리
+ 위임받은 모듈 (동적 추가)
```

**위임 범위에 따른 동적 메뉴 추가**

| 위임 범위 코드    | 추가되는 메뉴                      |
| ----------------- | ---------------------------------- |
| `risk_assessment` | 위험성평가                         |
| `compliance_docs` | 이행관리 / 업무문서관리 / 개선조치 |

### GNB 메뉴 렌더링 로직

```javascript
const BASE_MENUS = {
  principal_ceo_gm_sm_shm: [
    "dashboard",
    "tasks",
    "risk",
    "tbm",
    "inspection",
    "education",
    "policy",
    "budget",
    "opinion",
    "improvement",
    "compliance",
    "documents",
    "process",
    "contractor",
    "workplace",
    "support",
    "myinfo",
  ],
  principal_wkr: ["dashboard", "tasks", "tbm", "opinion", "support", "myinfo"],
  subcontractor: ["tasks", "tbm", "opinion", "support", "myinfo"],
  consulting: ["tasks", "support", "myinfo"],
};

function getMenuList(context) {
  const { company_type, role, consulting_scope } = context;

  if (company_type === "subcontractor") return BASE_MENUS.subcontractor;

  if (company_type === "consulting") {
    const menus = [...BASE_MENUS.consulting];
    if (consulting_scope?.includes("risk_assessment"))
      menus.splice(1, 0, "risk");
    if (consulting_scope?.includes("compliance_docs")) {
      menus.splice(1, 0, "compliance", "documents", "improvement");
    }
    return menus;
  }

  if (role === "WKR") return BASE_MENUS.principal_wkr;
  return BASE_MENUS.principal_ceo_gm_sm_shm;
}
```

## 6. 메뉴 ID — 화면 ID 매핑

| `tasks`       | 내 업무      | TSK01-L              | `tasks WHERE assignee=본인 AND status!=DONE COUNT`           |
| `risk`        | 위험성평가   | RSK01-V              | —                                                            |
| `tbm`         | TBM          | TBM01-L              | —                                                            |
| `inspection`  | 안전점검     | INS01-V              | —                                                            |
| `education`   | 안전보건교육 | EDU01-V              | —                                                            |
| `policy`      | 안전경영방침 | POL01-L              | —                                                            |
| `budget`      | 안전보건예산 | BGT01-V              | —                                                            |
| `opinion`     | 의견청취     | OPN01-L              | `opinion_requests WHERE status=RECEIVED AND site=본인 COUNT` |
| `improvement` | 개선조치     | IMP01-L              | `improvements WHERE status!=DONE AND site=본인 COUNT`        |
| `compliance`  | 이행관리     | CMP01-V              | —                                                            |
| `documents`   | 업무문서관리 | DOC01-L              | —                                                            |

| `contractor`  | 도급관리     | CON01-L              | —                                                            |
| `workplace`   | 사업장관리   | WRK01-L 또는 WRK01-D | —                                                            |
| `support`     | 고객지원     | SUP01-L              | —                                                            |
| `myinfo`      | 내 정보관리  | MYI01-S              | —                                                            |

## 9. 반응형 처리 (모바일)

| 구분     | 기준           | 처리                                               |
| -------- | -------------- | -------------------------------------------------- |
| 데스크탑 | 1024px 이상    | 사이드바 고정 표시                                 |
| 태블릿   | 768px ~ 1023px | 사이드바 아이콘만 표시 (레이블 숨김, 호버 시 툴팁) |
| 모바일   | 767px 이하     | 사이드바 숨김 + 하단 탭바로 대체                   |

**모바일 하단 탭바 구성**

컨텍스트별로 가장 자주 쓰는 메뉴 4~5개만 노출.

```
구독사업자 (SHM+)
[대시보드] [내 업무] [의견청취] [개선조치] [더보기]

근로자 / 도급업체
[내 업무] [TBM] [의견청취] [내 정보] [더보기]
```

## 10. 공통 레이아웃 사용 예외 화면
