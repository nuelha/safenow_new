### INS02-D 점검 상세

| 항목 | 내용 |
|------|------|
| 화면 ID | INS02-D |
| URL | `/safety-inspection/:id` |
| 접근 권한 | SHM+ |
| 진입 경로 | INS02-L 행 클릭 |

#### 섹션 구성

```
1. 점검 기본정보 (읽기전용)
   점검종류 | 점검명 | 점검일 | 점검자 | 장소
   출처배지 (의견청취 연동인 경우 표시)

2. 점검 실시 결과
   항목별: 점검항목 | 관련법령 | 결과(적합/부적합/해당없음)
   부적합 시: 부적합 내용 + 사진

3. 개선조치 연결 현황
   부적합 항목에서 생성된 개선조치 목록
   → IMP01-D 이동 링크
```

#### 점검 완료 처리 로직

```
점검 완료 버튼 클릭:
1. inspections.status = 'DONE'
2. 부적합 항목이 있는 경우:
   → improvements 테이블에 자동 INSERT (항목별)
     source_type = 'inspection'
     source_id = inspection_id
     status = 'PENDING'
3. 의견청취 연동인 경우:
   → opinion_requests 상태 업데이트 (점검중 → 개선조치 필요 여부 판단)
```

## 2. 안전보건교육 (EDU)

### EDU01-V 현황 탭

| 항목 | 내용 |
|------|------|
| 화면 ID | EDU01-V |
| URL | `/safety-education` |
| 접근 권한 | SHM+ |
| 기본 진입 화면 | ✅ |

#### 레이아웃

```
┌────────────────────────────────────────────────────┐
│ 안전보건교육                                        │
│ [현황] [교육목록]                                   │ ← 탭
│                                                    │
│ 안전보건교육 시간 합산 현황                         │
│ 기간선택: [2026년 상반기 ▼]                         │
│                                                    │
│ TBM 누적: Nh | 교육 누적: Nh | 합계: Nh            │
│                                                    │
│ 근로자별 이수현황 테이블                            │
│ 성명 | 부서 | TBM그룹 | TBM시간 | 교육시간 | 합계 | 상태 │
└────────────────────────────────────────────────────┘
```

#### 데이터 산출 로직

**시간 합산**

```
TBM 누적시간:
  SELECT SUM(duration_minutes) / 60
  FROM tbm_records tr
  JOIN tbm_group_members tgm ON tr.tbm_group_id = tgm.group_id
  WHERE tgm.user_id = :user_id
    AND tr.work_date BETWEEN :period_start AND :period_end
    AND tr.status = 'DONE'

교육 누적시간:
  SELECT SUM(hours)
  FROM education_records er
  WHERE er.user_id = :user_id
    AND er.education_date BETWEEN :period_start AND :period_end

합계 = TBM 누적시간 + 교육 누적시간
```

**근로자별 이수현황 상태 배지**

| 상태 | 조건 | 색상 |
|------|------|------|
| 이수완료 | 합계 >= 법정이수시간 | 초록 |
| 이수중 | 0 < 합계 < 법정이수시간 | 파랑 |
| 미이수 | 합계 = 0 | 회색 |

**기간 선택 옵션**

| 표시 | 기간 |
|------|------|
| 상반기 | 1월 1일 ~ 6월 30일 |
| 하반기 | 7월 1일 ~ 12월 31일 |

### EDU02-D 교육 상세/작성

| 항목 | 내용 |
|------|------|
| 화면 ID | EDU02-D |
| URL | `/safety-education/:id` |
| 접근 권한 | SHM+ |
| 진입 경로 | EDU02-L 행 클릭 / 내 업무에서 진입 |

#### 섹션 구성

```
1. 문서 정보 (읽기전용 헤더)

2. 교육 기본정보
   교육일 / 시작시간 / 종료시간 / 교육명

3. 강사 정보
   강사 입력방식: 조직도에서 선택 / 직접 입력
   강사명

4. 교육 내용
   교육내용 (TEXTAREA)
   특기사항 (TEXTAREA)

5. 참석자 전자서명 관리 (토글)
   ON 시: 서명 대상 / 서명 기간 설정
   참석자별 서명 현황 (전체/완료/미완료)
   - 서명 요청 → signature_requests 자동 생성
   - 참석자 TSK04-L에 자동 배정
```

#### 교육 완료 처리 로직

```
업무 저장(완료) 클릭:
1. education_schedules.status = 'DONE'
2. education_records 테이블에 참석자별 INSERT
   - user_id: 참석자
   - source_id: education_schedule_id
   - hours: 교육시간
3. EDU01-V 합산에 자동 반영
```
