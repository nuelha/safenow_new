# 도급관리 수급업체 연결 플로우 — 개발 스펙

> 작성일: 2026-04-28
> 관련 메뉴: 도급관리
> 우선순위: 신규 설계

---

## 1. 개요

### 문제

기존 구조는 수급업체가 안전NOW에 미리 등록되어 있어야 선정평가를 진행할 수 있다.
현실에서는 선정평가를 먼저 진행하고, 적격 업체가 확정된 후에 서비스에 초대하는 것이 자연스러운 흐름이다.

### 목표

- 수급업체가 안전NOW에 가입하지 않은 상태에서도 선정평가 진행 가능
- 선정 완료 후 해당 업체 담당자에게 가입 초대 알림톡 자동 발송
- 업체 가입/로그인 후 자동 소속 연결

---

## 2. 전체 플로우

```
STEP 1. 계약 기본정보 등록
   사업장 / 사업명 / 계약기간 / 업종 / 예산 / 계약내용
   → 저장 시 자동으로 STEP 2 진입

STEP 2. 수급업체 안전평가
   업체 추가
   ├── [가입 업체 검색] → 업체명 또는 사업자번호 검색 → 즉시 연결
   └── [신규 업체 입력] → 업체명 + 사업자번호 + 담당자 연락처 입력
                        → 미연결 상태로 평가 진행 가능

   평가 진행 (업체별)
   → 전체 업체 평가 완료 시 자동으로 STEP 3 진입

STEP 3. 적격업체 선정 + 결재
   적격 업체 체크 → 결재 자동 상신 → 결재 완료 시 자동으로 STEP 4 진입

STEP 4. 가입 초대 + 서약 (결재 완료 시 자동 실행)
   미가입 업체 → 담당자 연락처로 초대 알림톡 자동 발송
   기가입 업체 → 담당자에게 소속 추가 알림톡 자동 발송

   업체 담당자 처리
   ├── 미가입자: 링크 클릭 → 본인인증 → 가입 → 자동 소속 연결
   └── 기가입자: 링크 클릭 → 로그인 → 소속 추가 수락 → 자동 소속 연결

   연결 완료 후 → 서약서 알림톡 자동 발송 → 서약 완료 → 계약 활성화
```

## 2-1. STEP 2 평가서 선택

### 위치

STEP 2 진입 시 화면 최상단에 **평가서 선택 영역** 노출. 평가서가 선택되어야 업체별 평가하기 버튼이 활성화된다.

### 평가서 출처

업무문서관리(`documents.html`)에 등록된 문서 중 **카테고리=도급·계약**, **문서유형=체크리스트 또는 평가표**에 해당하는 문서.

### 평가서 선택 모달

```
[ 평가서 선택 ]

(검색창)

┌──────────────────────────────────────────────┐
│ ○ 수급업체 안전보건 평가표 (표준) [표준][사용중]│
│   도급·계약 · 항목 12개 · 100점 · 적격 80점   │
├──────────────────────────────────────────────┤
│ ○ 건설공사 수급업체 안전보건 평가표 [건설용]  │
│   도급·계약 · 항목 15개 · 100점 · 적격 75점   │
├──────────────────────────────────────────────┤
│ ○ 용역·위탁 업체 평가표 (간소형) [용역용]     │
│   도급·계약 · 항목 8개 · 100점 · 적격 70점    │
└──────────────────────────────────────────────┘

→ 업무문서관리에서 평가서 등록 →
```

### 평가서 메타데이터

평가서 문서에는 다음 메타데이터를 포함한다. 업무문서 등록 모달에서 **문서 유형 = `평가표` 또는 `체크리스트`** 선택 시 전용 입력 영역(`#evalMetaSection`) 노출.

| 필드            | 타입    | 비고                                                                    |
| --------------- | ------- | ----------------------------------------------------------------------- |
| 만점            | int     | 기본 100                                                                |
| 적격기준점수    | int     | 평가 결과 자동 판정에 사용 (기본 80)                                    |
| 평가척도        | enum    | `5단계` (매우우수~매우미흡) / `3단계` / `2단계` / `직접입력`             |
| 평가항목        | JSON    | 카테고리(이름·배점) → 항목(이름·설명·배점) 트리 구조                    |

### 평가서 등록 폼 — 항목 빌더

```
┌ 평가서 설정 ────────────────────────────────────┐
│  만점 [100]점   적격기준 [80]점 이상   척도 [5단계]│
│                                                 │
│  ▣ 평가 항목       항목 합계: 0 / 만점 100점     │
│                                                 │
│   #1 [카테고리명]            [배점]점  [🗑]      │
│      [항목명] [설명] [배점]점 [✕]                │
│      + 항목 추가                                 │
│   + 카테고리 추가                                │
└─────────────────────────────────────────────────┘
```

검증:
- 항목 배점 총합 = 만점 (실시간 검증, 일치 시 ✓ 표시 / 불일치 시 ⚠ 안내)
- 카테고리 배점은 해당 카테고리의 항목 배점 합과 같아야 함
- 카테고리 최소 1개 유지

### 평가서 변경 정책

- 평가가 진행 중이거나 일부 업체 평가가 완료된 상태에서 평가서를 변경하면 기존 평가 결과는 초기화 (확인 다이얼로그 노출)
- STEP 3 결재 상신 이후에는 평가서 변경 불가

## 2-2. 평가 진행 UI

### 트리거

업체 행의 `평가하기` 또는 `평가수정` 버튼 클릭 시 모달로 진입.

### 화면 구성

```
[ 수급업체 안전보건 평가 — (주)업체명 · 평가서: 수급업체 안전보건 평가표 ]

┌──────────────────────────────────────┐
│  현재 합계 점수    │   적격 기준 80점 │
│   88 / 100         │   [적격]         │
└──────────────────────────────────────┘

각 항목별 평가 척도 선택 (매우우수 100% · 우수 80% · 보통 60% · 미흡 40% · 매우미흡 20%)

▼ 1. 안전보건 경영시스템 (30점)
   안전보건 방침 및 목표 수립 (10점) [매우우수][우수][보통][미흡][매우미흡]
   안전보건 조직 운영        (10점) [...]
   안전보건 예산 편성        (10점) [...]

▼ 2. 산업재해 발생 이력 (25점)
   최근 3년 사망사고  (15점) [...]
   최근 3년 재해율    (10점) [...]

▼ 3. 안전보건 활동 (30점)
   위험성평가 실시        (10점) [...]
   안전보건 교육          (10점) [...]
   보호구 지급·관리       (10점) [...]

▼ 4. 법규 준수 (15점)
   산업안전보건법 위반 이력      (10점) [...]
   안전보건관리계획서 제출       (5점)  [...]

종합 의견 / 특이사항: [textarea]

평가 근거자료 첨부: [드래그앤드롭 영역]

[취소] [임시저장] [평가 완료]
```

### 점수 자동 계산

- 각 항목의 점수 = `배점 × 평가척도 비율`
- 척도 비율: 매우우수 100% / 우수 80% / 보통 60% / 미흡 40% / 매우미흡 20%
- 합계 점수는 입력 즉시 상단 카드에 반영
- 적격 자동 판정:
  - 80점 이상 → `적격`
  - 60~79점 → `조건부 (재평가 필요)`
  - 60점 미만 → `부적격`

### 평가 완료 후

- STEP 2 업체 목록의 `평가점수`, `결과` 컬럼이 즉시 업데이트
- 전체 업체 평가 완료 시 STEP 3 진행 가능 표시

#### 탭 A. 가입 업체 검색

안전NOW에 컨설팅업체 유형으로 가입된 업체 검색

| 필드   | 타입       | 비고                   |

---

#### 탭 B. 신규 업체 입력

안전NOW 미가입 업체. 평가에 필요한 최소 정보만 입력.

| 필드               | 타입        | 필수 | 비고                                       |
| ------------------ | ----------- | ---- | ------------------------------------------ |

| 사업자등록번호     | text        | ✅   | 중복 체크 (이미 가입된 업체면 탭 A로 안내) |
| 대표자명           | text        | ✅   |                                            |
| 담당자명           | text        | ✅   | 알림톡 수신자                              |
| 담당자 연락처      | tel         | ✅   | 알림톡 발송에 사용                         |
| 담당자 이메일      | email       | ❌   | 이메일 초대 병행 시 사용                   |
| 업종               | text/select | ✅   |                                            |
| 안전보건관리계획서 | file        | ❌   | 평가 전 제출 요청용                        |

## 4. STEP 4 초대 발송 상세

### 트리거

결재 완료 시 자동 실행. 선정된 업체 중 `미연결` 또는 `pending_invite` 상태인 업체에만 발송.

### 알림톡 내용 (미가입 업체)

```
[안전NOW] 수급업체 선정 안내

안녕하세요, {담당자명}님.

{사업장명}에서 수급업체로 선정되어 안내드립니다.

아래 링크를 통해 안전NOW에 가입하시면
계약 및 서약 절차가 자동으로 진행됩니다.

▶ 가입하기: {초대 링크}

링크 유효기간: 2주 ({만료일자})

문의: {사업장 담당자 연락처}
```

### 알림톡 내용 (기가입 업체)

```
[안전NOW] 소속 추가 요청

안녕하세요, {담당자명}님.

{사업장명}에서 수급업체 연결을 요청했습니다.

아래 링크를 통해 로그인 후 수락해 주세요.

▶ 수락하기: {초대 링크}

링크 유효기간: 2주 ({만료일자})
```

### 초대 링크 처리 로직

```
링크 클릭
├── 미가입자
│   → 가입 화면 (본인인증 → 부가정보 → 가입 완료)
│   → 가입 완료 시 해당 사업장 자동 소속 연결
│   → 업체 상태: pending_invite → 가입 완료 → 연결됨
│
└── 기가입자
    → 로그인 화면 (또는 로그인 상태면 바로 수락 화면)
    → 소속 추가 수락
    → 업체 상태: pending_invite → 연결됨
```

### 초대 링크 유효기간

- 기본: 2주
- 만료 시 재발송 버튼 노출 (관리자 수동 재발송)
- 재발송 가능 횟수 제한 없음

## 9. 구현 시 주의사항

1. **신규 입력 업체 데이터 보존**: 선정평가 전에 입력한 업체 기본 정보(업체명, 사업자번호 등)는 나중에 실제 안전NOW 계정과 연결될 때 매칭 기준으로 사용. 사업자번호 기준으로 매칭 권장.

2. **계정 연결 시점 데이터 동기화**: 미가입 업체가 가입 완료 후 소속 연결 시, 입력해둔 기본 정보(업체명, 사업자번호)와 실제 계정 정보를 비교하여 불일치 시 관리자에게 알림.

3. **정기평가/종료평가 자동 생성**: 계약 활성화(서약 완료) 시 정기평가 주기 기준으로 평가 스케줄 자동 생성. 이 때 업체는 반드시 `연결됨` 상태여야 함.

4. **미연결 업체의 정기평가**: 연결되지 않은 업체는 정기평가 대상에서 제외. 연결 완료 시 스케줄 자동 생성.

5. **안전NOW 미가입 업체의 서약**: 가입 완료 후 자동으로 서약 알림톡 발송. 가입 전 서약 불가.
