#### 레이아웃

```
┌────────────────────────────────────────────────────┐
│ 기간 탭: [26년 1분기] [26년 2분기] [25년 4분기]     │
│                                                    │
│ 달성률: 78%                                        │
│ 법정 이수시간: 12h | 정기교육 이수: 7h | TBM 인정: 2h│
│                                                    │
│ 이수증 출력                                         │
│ [전체 이수내역 PDF 출력]                            │
│                                                    │
│ 교육 이수 목록                                     │
│ 교육명 | 유형 | 교육일 | 시간 | 이수증 |            │
│ TBM 이수 내역                                      │
│ TBM그룹 | 날짜 | 시간 |                            │
└────────────────────────────────────────────────────┘
```

#### 달성률 계산

```
총 이수시간 = 정기교육 이수시간 + TBM 인정시간

정기교육 이수시간:
  SELECT SUM(hours) FROM education_records
  WHERE user_id = :user_id
    AND education_date BETWEEN :period_start AND :period_end

TBM 인정시간:
  SELECT SUM(duration_minutes) / 60 FROM tbm_records
  JOIN tbm_group_members ON tbm_group_id
  WHERE user_id = :user_id
    AND work_date BETWEEN :period_start AND :period_end
    AND status = 'DONE'

법정 이수시간:
  SELECT required_hours FROM legal_education_requirements
  WHERE industry_code = :industry_code
    AND employee_role = :role

달성률 = MIN(총이수시간 / 법정이수시간 × 100, 100)%
```

#### 이수증 출력

| 버튼 | 동작 |
|------|------|
| 교육별 이수증 | 해당 교육 PDF 생성 다운로드 |
| 전체 이수내역 PDF 출력 | 해당 기간 전체 이수내역 PDF 생성 |

**PDF 포함 내용**

```
회사명 / 사업장명 / 성명 / 교육기간
교육 이수 목록 (교육명 / 교육일 / 시간 / 강사)
TBM 인정 시간
총 이수시간 / 법정이수시간 / 달성률
발행일 / 발행자 (사업장관리자)
```
