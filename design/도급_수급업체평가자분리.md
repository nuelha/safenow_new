# 도급관리 — 수급업체 평가 / 평가자 분리 개발 스펙

> 작성일: 2026-04-29
> 관련 메뉴: 도급관리 > STEP 2. 수급업체 평가
> 우선순위: 수정 개발

---

## 1. 개요

### 문제

현재 STEP 2 수급업체 평가 화면에서 계약 담당자가 업체 추가와 평가를 모두 직접 진행하는 구조.
실제 업무에서는 계약 담당자와 평가 담당자(안전 부서)가 다른 경우가 일반적.

### 목표

- 계약 담당자: 업체 추가 + 평가자 지정 + 전체 진행 모니터링
- 안전 담당자(평가자): 내 업무에서 배정된 평가를 진행

---

## 2. 역할 분리

| 역할                 | 담당 업무                                                       |
| -------------------- | --------------------------------------------------------------- |
| 계약 담당자          | 계약 기본정보 등록, 업체 추가, 평가자 지정, 전체 진행 현황 조회 |
| 평가자 (안전 담당자) | 실제 안전보건 수준 평가 작성 및 완료 처리                       |

---

## 3. STEP 2 화면 변경사항

### 3-1. 업체 추가 모달 — 평가자 지정 필드 추가

기존 업체 추가 모달에 아래 필드 추가.

| 필드   | 타입        | 필수 | 비고                                     |
| ------ | ----------- | ---- | ---------------------------------------- |
| 평가자 | user search | ✅   | 조직도 내 검색. 안전보건관리자 권한 이상 |

저장 시 동작:

1. 수급업체 레코드 생성
2. **지정된 평가자의 내 업무에 평가 업무 자동 배정**
3. 평가자에게 알림톡 발송

---

### 3-2. 수급업체 평가 목록 컬럼 변경

기존 `담당자` 컬럼을 `평가자` 컬럼으로 교체.

| 컬럼       | 내용                    | 변경             |
| ---------- | ----------------------- | ---------------- |
| 업체명     |                         | 유지             |
| 사업자번호 |                         | 유지             |
| 대표자     |                         | 유지             |
| ~~담당자~~ | 수급업체 담당자         | **제거**         |
| 관리계획서 | 제출/미제출             | 유지             |
| **평가자** | 지정된 내부 평가 담당자 | **신규**         |
| 평가점수   | 완료 전: —              | 유지             |
| 결과       | 적격/부적격/대기        | 유지             |
| 상태       | 완료/진행중/대기        | 유지             |
| 관리       |                         | 변경 (아래 참조) |

**관리 컬럼 버튼 변경**

| 상태                      | 기존 버튼 | 변경 버튼                                              |
| ------------------------- | --------- | ------------------------------------------------------ |
| 평가자 미지정             | 평가하기  | **평가자 지정**                                        |
| 평가 대기 (평가자 지정됨) | 평가하기  | **평가 진행중** (비활성, 클릭 불가)                    |
| 평가 완료                 | 평가수정  | **평가수정** (계약담당자는 조회만, 평가자만 수정 가능) |

> 계약 담당자는 직접 평가 버튼에 접근 불가. 평가는 평가자 본인만 진행.

---

### 3-3. STEP 2 진행 조건 변경

```
전체 업체 평가 완료 (지정된 평가자 전원 완료 처리) 시 → STEP 3 진행 가능
미완료 업체 있으면 → 하단 경고 유지
  예: "전체 업체 평가가 완료되어야 다음 단계로 진행할 수 있습니다. (현재 2/3 완료)"
```

---

## 4. 평가자(안전 담당자) 업무 흐름

### 내 업무 배정

업체 추가 + 평가자 지정 저장 시 → 평가자의 **내 업무 > 문서업무** 탭에 자동 생성.

| 내 업무 항목 | 내용                                              |
| ------------ | ------------------------------------------------- |
| 업무명       | `[도급평가] {계약명} — {업체명}`                  |
| 업무 유형    | 도급안전평가                                      |
| 기한         | 계약 기본정보의 계약 시작일 기준 (또는 별도 지정) |
| 상태         | 대기                                              |

### 평가 진행 화면

평가자가 내 업무에서 해당 항목 클릭 시 평가 화면으로 진입.

평가 화면 구성:

- 업체 기본정보 (읽기전용)
- 관리계획서 파일 조회
- 평가서 항목별 점수 입력
- 평가결과 (적격/부적격) — 점수 기반 자동 판정 또는 수동 선택
- 평가의견 입력 (선택)
- 평가 완료 버튼

### 평가 완료 시 동작

1. 해당 업체 상태 → `완료`
2. 내 업무 해당 항목 → `완료` 처리
3. 계약 담당자에게 완료 알림톡 발송
4. 전체 업체 완료 시 → STEP 3 진행 가능 상태로 자동 전환

## 5. 알림톡 발송 정의

| 트리거           | 수신자      | 내용                                                                                  |
| ---------------- | ----------- | ------------------------------------------------------------------------------------- |
| 평가자 지정 저장 | 평가자      | `[안전NOW] {계약명} 수급업체 안전평가를 진행해 주세요. 업체: {업체명} / 기한: {날짜}` |
| 평가 완료        | 계약 담당자 | `[안전NOW] {업체명} 안전평가가 완료되었습니다. 결과: 적격 {점수}/100`                 |
| 전체 평가 완료   | 계약 담당자 | `[안전NOW] 전체 수급업체 평가가 완료되었습니다. STEP 3 선정을 진행해 주세요.`         |

## 6. STEP 3 선정 — 다수 업체 선정 지원

선정평가 결과 적격 업체가 여러 곳인 경우 다중 선정 가능.

### 선정 화면 구성

```
업체별 평가 결과 목록

☑  (주)안전건설    123-45-67890    적격 92점    선정
☑  대한설비(주)    234-56-78901    적격 88점    선정
☐  동방기계(주)    678-90-12345    부적격 61점  선정불가 (비활성)

→ 선정 완료 버튼 클릭
→ 체크된 업체 전체 결재 1건으로 일괄 상신
→ 결재 완료 후 선정된 업체 모두에게 초대 알림톡 일괄 발송
```

### 선정 규칙

- 부적격 업체: 체크박스 비활성화 (선정 불가)
- 적격 업체: 기본값 전체 체크, 개별 해제 가능
- 1개 이상 선택 후 선정 완료 가능
- 선정된 업체 전체에 결재 1건으로 일괄 상신


## 8. 구현 시 주의사항

1. **평가자 권한**: 평가자는 본인에게 배정된 업체만 평가 가능. 다른 업체의 평가 화면 접근 불가.

2. **계약 담당자 조회 권한**: 계약 담당자는 모든 업체의 평가 결과를 조회할 수 있지만 수정 불가.

3. **평가자 변경**: 평가 시작 전에는 계약 담당자가 평가자 변경 가능. 평가 진행 중에는 변경 불가 (또는 변경 시 기존 평가 초기화 경고).

4. **평가자 미지정 시**: 평가자가 지정되지 않은 업체는 STEP 3 진행 조건에서 제외. 목록에 `평가자 지정` 버튼 표시.

5. **다수 업체 선정 시 결재**: 결재 문서에 선정된 전체 업체 목록과 각 업체 평가 결과 포함.
