## 1. 안전경영방침 (POL)

### POL01-L 방침관리 탭

| 항목 | 내용 |
|------|------|
| 화면 ID | POL01-L |
| URL | `/safety-policy` |
| 접근 권한 | 전체 열람 / CEO 등록 |
| 기본 진입 화면 | ✅ |

#### 레이아웃

```
┌────────────────────────────────────────────────────┐
│ 안전경영방침                                        │
│ [방침관리] [게시알림] [목표·KPI] [경영방침 점검표]   │ ← 탭
│                                                    │
│ 버튼: [엑셀 다운로드] [+ 등록] (CEO만 노출)          │
│                                                    │
│ 번호 | 버전 | 구분 | 제·개정일자 | 사유 | 첨부       │
│ 결재 | 작성자 | 관리                                │
└────────────────────────────────────────────────────┘
```

#### safety_policies 테이블 주요 컬럼

| 컬럼 | 타입 | 설명 |

### POL01-F 방침 등록

| 항목 | 내용 |
|------|------|
| 화면 ID | POL01-F |
| URL | `/safety-policy/new` |
| 접근 권한 | CEO |
| 진입 경로 | POL01-L > + 등록 버튼 |

#### 입력 항목

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 버전 | TEXT | ✅ | 자동 생성 (수정 가능) |
| 구분 | RADIO | ✅ | 제정 / 개정 |
| 안전보건목표 | 반복 입력 | ✅ | 목표 항목 추가 |
| 세부방침 | 반복 입력 | ✅ | 방침 항목 추가 |
| 제·개정일자 | DATE | ✅ | |
| 시행일자 | DATE | ✅ | |
| 사유 | TEXTAREA | 조건부 ✅ | 개정 시 필수 |
| 첨부파일 | FILE | ❌ | |
| 서명 알림 | TOGGLE | — | 제정: 기본 ON / 개정: 기본 OFF |
| 서명 대상 | MULTI_SELECT | 서명ON 시 ✅ | |
| 서명 기한 | DATE | 서명ON 시 ✅ | |

#### 저장 동작

```
저장 클릭:
1. safety_policies INSERT
2. 서명 알림 ON인 경우:
   → signature_requests 테이블에 대상자별 INSERT
   → 대상자 TSK04-L(전자서명 탭)에 자동 배정
   → 알림톡 발송: "[안전NOW] {버전} 안전경영방침 서명 요청"
3. POL02-L (게시알림 탭)에 자동 반영
```

## 2. 안전보건예산 (BGT)

### BGT01-V 예산 현황

| 항목 | 내용 |
|------|------|
| 화면 ID | BGT01-V |
| URL | `/safety-budget` |
| 접근 권한 | CEO·GM (전체 집계) / SM·SHM (담당 사업장만) |
| 기본 진입 화면 | ✅ |

#### 레이아웃

```
┌────────────────────────────────────────────────────┐
│ 안전보건예산                                        │
│                                                    │
│ 연도 선택: [2026 ▼]                                │
│                                                    │
│ [CEO·GM 뷰] 전체 사업장 집계 테이블                 │
│ 사업장 | 예산 | 집행 | 집행율                       │
│ (클릭 시 해당 사업장 상세로 이동)                    │
│                                                    │
│ [SM·SHM 뷰] 담당 사업장 예산 상세 (직접 노출)        │
│                                                    │
│ 예산 상세 테이블                                    │
│ 항목 | 전년예산 | 전년실적 | 올해예산                │
│ 상반기 | 하반기 | 집행금액 | 집행율                 │
│                                                    │
│ [인쇄] [항목관리] [저장] [결재상신]                  │
│                                                    │
│ 결재 이력                                          │
└────────────────────────────────────────────────────┘
```

#### safety_budgets 테이블 주요 컬럼

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | UUID PK | |
| `site_id` | UUID FK | |
| `year` | YEAR | |
| `items` | JSON | 예산 항목 배열 |
| `approval_status` | ENUM(`DRAFT`, `PENDING`, `APPROVED`) | |
| `approved_at` | TIMESTAMP | |

#### 예산 항목 JSON 구조

```json
[
  {
    "item_name": "보호구 구입",
    "prev_budget": 5000000,
    "prev_actual": 4800000,
    "current_budget": 6000000,
    "first_half": 3000000,
    "second_half": 3000000,
    "executed": 0
  }
]
```

#### 집행율 자동 계산

```
집행율 = 집행금액 / 올해예산 × 100 (%)
```

#### 결재 상신 로직

```
결재상신 클릭:
1. safety_budgets.approval_status = 'PENDING'
2. approvals 테이블에 결재 레코드 생성
   approval_type: 'budget'
3. 결재라인: SM → GM → CEO
4. 각 결재자 내 업무 결재대기에 배정
5. 결재 완료 시 approval_status = 'APPROVED'
```

#### 항목관리 모달

- 예산 항목 추가/수정/삭제
- 항목명 + 상반기/하반기 예산 배분 입력
