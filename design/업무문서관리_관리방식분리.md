# 업무문서관리 — 데이터입력형 문서 유형 정의

> 작성일: 2026-04-29
> 관련 문서: 업무문서관리*관리방식분리*개발스펙.md
> 우선순위: 수정 개발

---

## 1. 개요

데이터입력형 문서 등록 시 **문서 유형**을 선택하면, 해당 유형에 맞는 기본 섹션과 항목이 자동으로 구성됨.
사용자는 기본 구성을 그대로 사용하거나 항목을 추가/수정/삭제할 수 있음.

---

## 2. 데이터입력형 문서 유형 목록 (5종)

| 유형       | 설명                                                             | 주요 특징                              | 사용처                            |
| ---------- | ---------------------------------------------------------------- | -------------------------------------- | --------------------------------- |
| 평가표     | 항목별 점수를 매겨 적격 여부를 판단하는 문서                     | 점수 자동 합산 + 적격/부적격 자동 판정 | 도급 선정/정기/종료평가, 인력평가 |
| 회의록     | 회의 내용과 결정사항을 기록하는 문서                             | 참석자 서명 연동                       | 안전보건위원회 회의록             |
| 기록부     | 반복적으로 발생하는 사건/행위를 테이블 형태로 누적 기록하는 문서 | 행 반복 추가                           | 교육기록부, 점검기록              |
| 확인서     | 특정 내용을 확인했음을 증명하는 문서                             | 서명 필수                              | 안전수칙 확인서, 서약서           |
| 체크리스트 | 항목별로 적합/부적합/해당없음을 체크하는 문서                    | 부적합 시 개선조치 자동 연동           | 안전점검 점검지                   |

## 3. 유형별 기본 구성 상세

---

### 3-1. 평가표

**특징**

- 섹션별로 항목 구성 가능 (카테고리별 배점 관리)
- 항목별 점수 입력 시 합계 자동 계산
- 만점 대비 달성 점수에 따라 적격/부적격 자동 판정
- 도급관리, 인력평가 등 다른 메뉴에서 양식으로 선택 가능

**필수 설정**

| 설정          | 내용                                                                                   |
| ------------- | -------------------------------------------------------------------------------------- |
| 만점          | 전체 배점 합계 (예: 100점)                                                             |
| 적격 기준점수 | 이 점수 이상이면 적격 (예: 80점 이상)                                                  |
| 평가 척도     | 5단계(매우우수~매우미흡) / 3단계(우수/보통/미흡) / 2단계(적합/부적합) / 직접 점수 입력 |

**기본 섹션 구성**

```
섹션 1. 평가 기본정보
  - 피평가자/업체명      [텍스트, 필수]
  - 평가일자             [날짜, 필수]
  - 평가자               [사용자 선택, 필수]

섹션 2. 평가 항목 (반복 추가 가능)
  - 항목명               [텍스트, 필수]
  - 배점                 [점수, 필수]
  - 평가결과             [척도 선택에 따라 자동 생성]
  - 취득점수             [자동 계산]
  - 비고                 [텍스트, 선택]

섹션 3. 종합 의견 (선택)
  - 종합 평가의견        [텍스트(장문), 선택]
  - 개선 권고사항        [텍스트(장문), 선택]
```

**자동 계산 항목**

- 섹션별 소계
- 전체 합계
- 만점 대비 달성률 (%)
- 적격/부적격 자동 판정

### 3-2. 회의록

**특징**

- 안건별 협의 내용을 구조화하여 기록
- 참석자 목록 + 전자서명 연동
- 결정사항 별도 섹션으로 관리

**기본 섹션 구성**

```
섹션 1. 회의 기본정보
  - 회의명               [텍스트, 필수]
  - 회의일시             [날짜+시간, 필수]
  - 회의 장소            [텍스트, 필수]
  - 회의 목적/안건 요약  [텍스트(장문), 선택]

섹션 2. 참석자 (반복 행)
  - 소속/부서            [텍스트]
  - 성명                 [텍스트 또는 사용자 선택]
  - 직책                 [텍스트]
  - 서명                 [전자서명, 선택]

섹션 3. 안건 (반복 추가 가능)
  - 안건 번호            [자동 증가]
  - 안건명               [텍스트, 필수]
  - 협의 내용            [텍스트(장문), 필수]
  - 결정사항             [텍스트(장문), 선택]
  - 담당자               [텍스트, 선택]
  - 완료 예정일          [날짜, 선택]

섹션 4. 첨부자료 (선택)
  - 첨부파일             [파일첨부, 선택]
```

### 3-3. 기록부

**특징**

- 동일한 항목이 반복적으로 발생하는 행위를 시간 순서로 누적 기록
- 테이블(스프레드시트) 형태로 표시
- 기간별 조회 및 엑셀 내보내기 가능

**기록부 유형별 기본 열 구성**

#### 교육기록부

| 열        | 타입       | 필수 |
| --------- | ---------- | ---- |
| 교육일시  | 날짜+시간  | ✅   |
| 교육명    | 텍스트     | ✅   |
| 교육 내용 | 텍스트     | ✅   |
| 강사      | 텍스트     | ✅   |
| 교육 시간 | 숫자(시간) | ✅   |
| 교육 장소 | 텍스트     | ✅   |
| 참여 인원 | 숫자       | ✅   |
| 비고      | 텍스트     | ❌   |

#### 점검기록부

| 열        | 타입                    | 필수             |
| --------- | ----------------------- | ---------------- |
| 점검일시  | 날짜+시간               | ✅               |
| 점검 종류 | 선택(일상/주간/특별)    | ✅               |
| 점검자    | 텍스트 또는 사용자      | ✅               |
| 점검 장소 | 텍스트                  | ✅               |
| 점검 결과 | 선택(이상없음/이상있음) | ✅               |
| 이상 내용 | 텍스트                  | 이상있음 시 필수 |
| 조치 내용 | 텍스트                  | ❌               |
| 비고      | 텍스트                  | ❌               |

**공통 기본 설정**

```
섹션 1. 기록부 기본정보 (헤더)
  - 기록부명             [자동 (문서명에서 가져옴)]
  - 사업장               [자동 (현재 사업장)]
  - 작성 기간            [날짜 범위, 자동 계산]

섹션 2. 기록 테이블
  - 열 구성은 유형에 따라 위 정의 참조
  - 행 추가 버튼으로 반복 입력

섹션 3. 요약 통계 (자동)
  - 전체 건수
  - 기간별 집계 (월별/분기별)
```

### 3-4. 확인서

**특징**

- 특정 내용을 확인/동의했음을 증명
- 전자서명 필수
- 서명 완료 시 PDF 자동 생성 및 잠금 (수정 불가)

**기본 섹션 구성**

```
섹션 1. 확인서 기본정보
  - 확인서 제목          [텍스트, 필수]
  - 발행일               [날짜, 필수]
  - 유효기간             [날짜, 선택]

섹션 2. 확인 내용
  - 확인 항목 (반복 추가)
    - 항목 번호          [자동 증가]
    - 내용               [텍스트(장문), 필수]
    - 확인 여부          [체크박스, 선택]

섹션 3. 서명
  - 확인자 성명          [텍스트 또는 사용자 선택, 필수]
  - 소속/부서            [텍스트, 선택]
  - 서명일               [날짜, 자동(서명 시점)]
  - 전자서명             [전자서명, 필수]
```

**서명 완료 후 동작**

- 문서 상태 → 서명완료 (잠금, 수정 불가)
- PDF 자동 생성
- 서명자에게 사본 알림톡 발송 (선택)

### 3-5. 체크리스트

**특징**

- 항목별로 적합/부적합/해당없음 체크
- 부적합 항목 발생 시 개선조치 자동 생성 연동
- 안전점검 메뉴의 점검지로 활용됨

**기본 섹션 구성**

```
섹션 1. 점검 기본정보
  - 점검명               [텍스트, 필수]
  - 점검일시             [날짜+시간, 필수]
  - 점검자               [사용자 선택, 필수]
  - 점검 장소            [텍스트, 필수]

섹션 2. 점검 항목 (반복 추가 가능)
  - 점검 항목명          [텍스트, 필수]
  - 관련 법령/기준       [텍스트, 선택]
  - 점검 결과            [선택: 적합/부적합/해당없음, 필수]
  - 부적합 내용          [텍스트, 부적합 시 필수]
  - 사진 첨부            [파일첨부, 선택]

섹션 3. 종합의견 (선택)
  - 종합 소견            [텍스트(장문), 선택]
  - 전체 평가            [선택: 양호/보통/불량, 선택]
```

**부적합 처리 연동**

- 점검 완료 시 부적합 항목 개수 자동 집계
- 개선조치 생성 여부 선택 팝업 노출
  - 생성 선택 시: 부적합 항목별 개선조치 자동 생성 → 개선조치 메뉴로 연결
  - 미생성 선택 시: 기록만 저장

## 4. 문서 유형별 다른 메뉴 연동 정리

| 문서 유형  | 연동 메뉴                        | 연동 조건                       |
| ---------- | -------------------------------- | ------------------------------- |
| 평가표     | 도급관리 > 선정/정기/종료평가    | 문서 유형 = 평가표              |
| 평가표     | 사업장관리 > 조직관리 > 인력평가 | 문서 유형 = 평가표              |
| 체크리스트 | 안전점검 > 점검지 관리           | 문서 유형 = 체크리스트          |
| 확인서     | 도급관리 > 서약서 관리           | 문서 유형 = 확인서              |
| 기록부     | 안전보건교육 > 교육기록          | 문서 유형 = 기록부 (교육기록부) |

## 5. 항목 유형 전체 정의

데이터입력형 문서에서 사용 가능한 항목 유형 목록.

| 유형 코드     | 표시명       | 설명                    | 사용 문서 유형 |
| ------------- | ------------ | ----------------------- | -------------- |
| `text`        | 텍스트       | 단행 텍스트 입력        | 공통           |
| `textarea`    | 텍스트(장문) | 여러 줄 텍스트          | 공통           |
| `number`      | 숫자         | 숫자 입력               | 공통           |
| `date`        | 날짜         | 날짜 선택               | 공통           |
| `datetime`    | 날짜+시간    | 날짜와 시간 선택        | 공통           |
| `select`      | 선택         | 단일 선택               | 공통           |
| `checkbox`    | 체크박스     | 예/아니오               | 확인서         |
| `score`       | 점수         | 숫자 + 만점 설정 + 척도 | 평가표         |
| `inspection`  | 점검결과     | 적합/부적합/해당없음    | 체크리스트     |
| `signature`   | 전자서명     | 서명 입력               | 확인서, 회의록 |
| `user`        | 사용자 선택  | 조직도 내 검색 선택     | 공통           |
| `file`        | 파일첨부     | 파일 업로드             | 공통           |
| `auto_number` | 자동 번호    | 행 추가 시 자동 증가    | 기록부, 회의록 |
| `auto_calc`   | 자동 계산    | 다른 항목 기반 계산     | 평가표         |

## 6. 문서 등록 플로우 — 유형 선택 시 분기

```
[데이터입력형 선택]
       ↓
문서 유형 선택
  ├── 평가표      → 평가표 기본 구성 로드 + 평가 설정 노출
  ├── 회의록      → 회의록 기본 구성 로드
  ├── 기록부      → 기록부 세부 유형 선택 (교육기록부 / 점검기록부 / 직접 구성)
  ├── 확인서      → 확인서 기본 구성 로드 + 서명 설정 필수
  └── 체크리스트  → 체크리스트 기본 구성 로드 + 개선조치 연동 설정 노출

       ↓
기본 구성 미리보기
사용자 항목 추가/수정/삭제 가능

       ↓
STEP 3. 문서 정보 입력 (공통)
카테고리 / 설명 / 법적근거 / 태그 / 버전 / 이행관리 연결
```
